Media queries


Uma regra é uma declaração iniciado com o caractere @ e termina com um ;
@charset "charset"; -UTF-8 e iso-8859-15
@import url("outro-arquivo.css"); - Utilizado para importar regras de outros arquivos CSS
@media - utilizado para restringir o escopo das folhas de estilo, diferenciar dispositivo (celular, tablet e monitores) @media (max-width: 600px) { … } possui restrições [altura e largura da janela; altura e largura do dispositivo; orientação (retrato e paisagem para dispositivos móveis); resolução]
@media not|only mediatype and (mediafeature and|or|not mediafeature) { Código CSS }
-> Mediatype: all | print | screen | speech
-> Media features: (max/min) height, width, resolution e orientation: landscape ou portrait


Display


O display define o comportamento do elemento html e a forma como é apresentado diferenciando-o de outros elementos, sendo possível modificar seus comportamentos.
Pode ser nehum (none), em linha (inline), em bloco (block) ou em linha/bloco (inline-block).
none é diferente de visibility: hidden -> none retira o elemento e o hidden apenas some com o elemento.
inline-block mostra elemento em linha e permite aplicar propriedades de bloco.
